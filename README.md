## Running the project

1.  Clone the repo:

    ```console
    $ git clone https://github.com/AliReza1083/Portfolio.git
    ```

2.  Navigate to the cloned directory:

    ```console
    $ cd Portfolio
    ```

3.  Install dependencies:

    ```console
    $ npm install
    ```
    ```console
    $ yarn
    ```

4.  Run the project

    ```console
    $ npm run dev
    ```
    ```console
    $ yarn dev
    ```
